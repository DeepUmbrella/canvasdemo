# canvasDemo

This is a game project running in the browser based on the canvas API written in typescript [Git Repository](https://github.com/DeepUmbrella/canvasdemo.git)

## how to start

```shell
# install dependence

   pnpm install

# start project

    # start for development
    pnpm run dev

    # start for development
    pnpm run start



```
